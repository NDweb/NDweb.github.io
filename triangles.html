5. Cвойство transform
Изображение Параметры Примечания
transform:
 none исходное изображение
transform:
 rotate(15deg) поворот по часовой стрелке на 15 град
transform:
 rotate(-15deg) поворот против часовой стрелки на 15 град
transform:
 scale (0.5, 1) уменьшение на 50% по горизонтали
transform:
 scale (1, 0.5) уменьшение на 50% по вертикали
transform:
 scale (0.5, 0.5) уменьшение на 50% по горизонтали и вертикали
transform:
 scale (1, 1.5) увеличение на 50% по вертикали
transform:
 scale (1.5, 1) увеличение на 50% по горизонтали
transform:
 skew (30, 0) наклон на 30 град по горизонтали
transform:
 skew (0, 15) наклон на 15 град по вертикали
transform:
 skew (15, 15) наклон на 15 град по горизонтали и вертикали
transform:
 skew (-15, -15) наклон на -15 град по горизонтали и вертикали
transform:
 translate (15рх, 0px) cмещение на 15рх по горизонтали
transform:
 translate (0рх, 5px) cмещение на 5рх по вертикали
transform:
 translate (15рх, -5px) cмещение на 15px по горизонтали и -5рх по вертикали
transform:
 matrix
(scale, skewY, skewX, scaleY, translateX, translateY);		 уменьшение масштаба по горизонтали scale -0.4,
наклон по горизонтали skewY 0.6,
наклон по вертикали skewX 1,
меньшение масштаба по вертикали scaleY 0.4,
смещение по горизонтали translateX -5px,
смещение по вертикали translateY 10px
transform:
 transform-origin (x, у, z),
для х: px % left center right,
для у: px % top center bottom,
для z: px		 координаты точки, относительно которой будет происходить трансформация элемента (смещение 10% по х, смещение 10% по y при rotate 10 град)
